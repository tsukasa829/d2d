# チャットスクリプト制作インストラクション

## 概要

このチャットは、**ユーザーの負の感情を解放する**ためのロールプレイング型セッションです。ユーザーはネガティブな感情を司る猫になりきり、自分の感情を表現することで心理的なカタルシスを得ます。

## 基本構造

### 各Dayのテーマ

各Dayは特定の感情をテーマにします：

- **Day 1**: 悲しみ猫
- **Day 2**: 怒り猫
- **Day 3**: 不安猫
- **Day 4**: 孤独猫
- **Day 5**: 嫉妬猫
- **Day 6**: 恥猫
- **Day 7**: 絶望猫

（※テーマは調整可能）

### セッションの流れ

1つのセッションは**5セット**で構成されます。
各セットは以下の**4つのメッセージ**から成ります：

#### 1. 相手役の猫が話しかける
- 悲しそうな/ネガティブなシチュエーションの話をする
- 例: 「さっきの集合写真、君が写ってなかったにゃ…ごめんにゃ…」
- 例: 「大事にしてたおもちゃ…壊れちゃったにゃ…」

#### 2. ユーザーの選択肢（3つ）
- **2つは外れ**: 感情を否定・抑圧する選択肢
  - 例: 「まぁどうせ写り悪いにゃ」
  - 例: 「気にしてないにゃ」
- **1つは正解**: テーマとなる感情を表現する選択肢
  - 例: 「みんなと写りたかったから…悲しかったにゃ」（悲しみを表現）
  - 例: 「壊れてショックだにゃ…大事だったのに」（悲しみを表現）

#### 3. 相手役の猫から了承/同意の返事
- ユーザーの感情を受け止める
- 例: 「そんな思いをさせてたにゃ…ごめんにゃ…」
- 例: 「そうだったんだにゃ…」

#### 4. ねこ先生（メンター）が感情を肯定
- ユーザーの感情表現を称賛・肯定する
- 例: 「悲しい気持ちを否定しないのはとても大事にゃ。よく言えたにゃ。」
- 例: 「自分の気持ちに正直になれたにゃ。素晴らしいにゃ。」

### セットの繰り返し

上記の4つのメッセージを**1セット**とし、これを**5セット繰り返します**。

各セットで異なるシチュエーションを提示し、ユーザーが繰り返し感情表現を練習できるようにします。

## チャットスクリプトの作成方法

### Front Matter 設定

```yaml
---
userAvatar: /avatars/sad.png         # テーマに応じたアバター
defaultBot: other                     # 相手役の猫をデフォルトに
requireCorrect: true                  # 正解を選ぶまで進めない
wrongMessage: "猫の気持ちになって、選んでみてください"
bots:
  other:
    displayName: 相手猫
    avatar: /avatars/otherCat.png
  teacher:
    displayName: ねこ先生
    avatar: /avatars/teacherCat.png
---
```

### メッセージの記述例

```markdown
Bot(other): さっきの集合写真、君が写ってなかったにゃ…ごめんにゃ…

User:
- x: まぁどうせ写り悪いにゃ
- x: 気にしてないにゃ
- o: みんなと写りたかったから…悲しかったにゃ

Bot(other): そんな思いをさせてたにゃ…ごめんにゃ…

Bot(teacher): 悲しい気持ちを否定しないのはとても大事にゃ。よく言えたにゃ。
```

これを5セット繰り返します。

### 最終セット後

最後のセット完了後、次のステージへ進むボタンを表示：

```markdown
Bot(teacher)[button]: 次へ進む | /stageup?nextStage=2
```

## LLMへの依頼フォーマット

チャットスクリプトを作成する際は、以下のようにLLMに依頼してください：

```
◯◯猫をDay◯に作って
```

**例:**
- 「悲しみ猫をDay1に作って」
- 「怒り猫をDay2に作って」
- 「不安猫をDay3に作って」

## 生成時の注意事項

1. **必ず5セット作成する**
   - 各セットは独立したシチュエーション
   - バリエーションを持たせる

2. **シチュエーションの多様性**
   - 人間関係、物の喪失、期待外れ、誤解、拒絶など
   - 日常的で共感しやすい場面

3. **選択肢の設計**
   - 外れ選択肢: 感情を否定・抑圧・逃避する内容
   - 正解選択肢: 感情を率直に表現する内容
   - 選択肢は明確に対比させる

4. **メンターの言葉**
   - 感情を肯定する
   - 表現したことを称賛する
   - 安全な場を提供する雰囲気

5. **Front Matter の設定**
   - `requireCorrect: true` を必ず設定
   - `wrongMessage: "猫の気持ちになって、選んでみてください"` を使用
   - アバター画像をテーマに合わせる

6. **語尾について**
   - 語尾に「にゃ」をつける必要はありません
   - 自然な日本語で記述してください

7. **ねこ先生の口調**
   - 「〜できました。」ではなく「〜できましたね。」のように、寄り添うような優しい語尾を使う
   - 例: 「怒りの感情を認めることができましたね。」
   - 例: 「自分の時間を大切にする怒りを表現できましたね。」

8. **ねこ先生の段階的な励まし**
   - 1セット目: 感情表現を肯定する
   - 2セット目以降: 「もっと怒ってもいいんですよ」「もっと表現していいんです」など、より深い表現を促す言葉を追加
   - 例: 「素晴らしいです。もっと怒ってもいいんですよ。」
   - 例: 「その怒りは正当です。どんどん表現していいんです。」

9. **猫のプロフィール紹介**
   - セッション開始前に、ねこ先生がテーマとなる猫を紹介する
   - アバター画像を表示し、猫の特徴と世界観を説明
   - ユーザーが安心して感情表現できる雰囲気を作る
   - 例: 「この子は怒り猫。怒るのが大好きな子です。猫の世界では、どんな感情も許されます。もちろん、怒りもOKです。怒り猫になったつもりで、会話してみましょう!」

## ファイル名規則

作成するファイル名は以下の通り：

```
docs/chats/day1.md
docs/chats/day2.md
docs/chats/day3.md
...
```

## 完全な例

以下は「悲しみ猫」（Day1）の完全な例です：

```markdown
---
userAvatar: /avatars/sad.png
defaultBot: other
requireCorrect: true
wrongMessage: "猫の気持ちになって、選んでみてください"
bots:
  other:
    displayName: 相手猫
    avatar: /avatars/otherCat.png
  teacher:
    displayName: ねこ先生
    avatar: /avatars/teacherCat.png
---

<!-- セット1 -->
Bot(other): さっきの集合写真、君が写ってなかったにゃ…ごめんにゃ…

User:
- x: まぁどうせ写り悪いにゃ
- x: 気にしてないにゃ
- o: みんなと写りたかったから…悲しかったにゃ

Bot(other): そんな思いをさせてたにゃ…ごめんにゃ…

Bot(teacher): 悲しい気持ちを否定しないのはとても大事にゃ。よく言えたにゃ。

<!-- セット2 -->
Bot(other): 大事にしてたおもちゃ…壊れちゃったにゃ…

User:
- x: そんなのいらないにゃ
- o: 壊れてショックだにゃ…大事だったのに
- x: 新しいの買えばいいにゃ

Bot(other): 大切にしてたのに…悲しいにゃね…

Bot(teacher): 大切なものを失った悲しみを表現できたにゃ。それが第一歩にゃ。

<!-- セット3 -->
Bot(other): 君を誕生日会に呼ぶの忘れてたにゃ…

User:
- x: 別に誕生日なんて興味ないにゃ
- o: 仲間外れにされて悲しかったにゃ…
- x: 忘れられるのは慣れてるにゃ

Bot(other): 本当にごめんにゃ…そんな思いさせたくなかったにゃ…

Bot(teacher): 寂しさを素直に認められたにゃ。とても勇気がいることにゃ。

<!-- セット4 -->
Bot(other): 君の作った料理、みんな残しちゃったにゃ…

User:
- x: 元々料理は得意じゃないにゃ
- x: 食べなくて結構にゃ
- o: 頑張ったのに…悲しいにゃ…

Bot(other): せっかく作ってくれたのに…ごめんにゃ…

Bot(teacher): 努力が報われなかった悲しみを言葉にできたにゃ。素晴らしいにゃ。

<!-- セット5 -->
Bot(other): 君のこと、みんなの前で笑っちゃったにゃ…ごめんにゃ…

User:
- x: 笑われるのは慣れてるにゃ
- o: 傷ついたにゃ…信じてたのに…
- x: 別にどうでもいいにゃ

Bot(other): 本当にごめんにゃ…君を傷つけるつもりはなかったにゃ…

Bot(teacher): 裏切られた悲しみを表現できたにゃ。これで君は一歩前に進んだにゃ。

Bot(teacher)[button]: 次へ進む | /stageup?nextStage=2
```

## まとめ

- **目的**: ユーザーの負の感情を解放する
- **形式**: ロールプレイング（ネガティブ感情を司る猫になりきる）
- **構造**: 4メッセージ × 5セット = 計20メッセージ（選択肢除く）
- **依頼方法**: 「◯◯猫をDay◯に作って」
- **出力**: `docs/chats/day◯.md` ファイル
