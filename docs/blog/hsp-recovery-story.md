# カウンセラーで治らなかった私がHSPを卒業した話

## はじめに｜カウンセリングでは変われなかった

こんにちは。元HSP（Highly Sensitive Person）で、会社員を辞めるほど追い詰められていた私が、今では普通に人間関係を楽しめるようになった体験をシェアします。

「HSPは生まれつきの気質だから、一生付き合っていくしかない」

カウンセラーにそう言われたとき、私は絶望しました。

## 会社員時代｜毎日が地獄だった

当時の私は、典型的なHSPでした。

- 上司の機嫌を常に気にして、顔色を伺う
- 誰かが怒られていると、自分が怒られているような気持ちになる
- 会議で発言するだけで心臓がバクバク
- 帰宅後は毎晩「1人反省会」で自分を責める
- 月曜日が来るのが怖くて、日曜夜は涙が出る

「私は社会に向いていない」と思い、メンタルを病んで出社できなくなりました。

## カウンセリングに通ったけれど…

退職後、藁にもすがる思いでカウンセリングに通い始めました。

「話を聞いてもらえる」だけで、少しは楽になりました。
でも、**根本的には何も変わりませんでした。**

- 「あなたは繊細なんですね」と共感してもらえる
- 「自分を大切にしましょう」とアドバイスをもらう
- でも、具体的に何をすればいいのかわからない
- セッションが終わると、また元の不安な自分に戻る

半年通って気づいたのは、**カウンセリングは「理解してもらう場所」であって、「変わる場所」ではない**ということでした。

## 転機｜心理学を自分で学び始めた

「このままじゃダメだ」

そう思った私は、自分で心理学を学び始めました。

- 認知行動療法
- 交流分析
- インナーチャイルド
- 愛着理論

たくさんの本を読み、オンライン講座を受けました。

そして、ある日気づいたんです。

**「HSPは気質じゃなくて、過去の体験が作った『反応パターン』なんだ」**

## 心理ロールプレイングとの出会い

心理学を学ぶ中で出会ったのが、アメリカで生まれた新しい心理手法**心理ロールプレイング**という手法でした。

これは、キャラクターになりきり、自分の感情を表現し、認めてもらうことです

例えば、私の場合：

### 幼少期の記憶

小学生の時、母親に「あなたはいつも迷惑ばかりかけて」と言われた記憶。

この時、私は「自分は存在するだけで迷惑な人間だ」と思い込んでしまいました。

### ロールプレイで「感情の成功体験を作る」

心理ロールプレイングでは、悲しみ猫や怒り猫など、様々な感情を持つキャラクターになりきります。

そして、**素直に感情を表現し、それを認めてもらう体験**を積み重ねていきます。

例えば、「悲しみ猫」になって：

「写真に写ってなくて、悲しかった」
「大事なおもちゃが壊れて、ショックだった」
「誕生日を忘れられて、寂しかった」

こんな風に、普段は我慢していた感情を、思いっきり表現します。

すると、相手は「そんな思いをさせてごめんね」と受け止めてくれる。

**これが「感情の成功体験」です。**

普通の生活では、感情を表現すると「そんなことで怒るなんて」「気にしすぎ」と否定されることが多い。

だから私たちは、感情を出すこと自体が怖くなってしまった。

でも、ロールプレイの世界では、どんな感情も許される。
悲しんでもいい。怒ってもいい。寂しがってもいい。

**「感情を出しても大丈夫」という成功体験を何度も積むことで、心が少しずつ安全を感じられるようになります。**

## 変化｜HSPの症状が消えていった

心理ロールプレイングを続けて3ヶ月。

気づけば、HSPの症状がほとんど消えていました。

**Before（HSP時代）**
- 人の顔色ばかり気にしていた
- 自分の意見が言えなかった
- 些細なことで傷ついていた
- 毎日疲れ果てていた

**After（HSP卒業後）**
- 人の機嫌は人の問題だと思えるようになった
- 自分の意見を素直に言えるようになった
- 他人の言葉を「そういう意見もあるね」と流せるようになった
- 毎日が楽しくなった

## なぜカウンセリングではダメだったのか？

今振り返ると、カウンセリングと心理ロールプレイングの違いは明確です。

**カウンセリング**
- 「今の気持ち」を聞いてもらう
- 理解と共感を得られる
- でも、過去の記憶は変わらない

**心理ロールプレイング**
- 「過去の記憶」に直接アプローチする
- トラウマ体験を「やり直す」
- 潜在意識レベルで書き換わる

HSPの根本原因は、**過去の体験で作られた「自分は傷つきやすい存在だ」という思い込み**です。

その思い込みを変えない限り、いくら話を聞いてもらっても変わらないんです。
